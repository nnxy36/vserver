LXC-Container sind Containervirtualisierung — leichter als vollständige KVM-VMs, aber für viele Anwendungsfälle absolut ausreichend. Webserver, Teamspeak, kleine Bots, Monitoring-Dienste — das läuft alles problemlos. Wichtig zu wissen: LXC-Pakete bei vServer.site sind IPv6-only, haben also keine IPv4-Adresse dabei.
